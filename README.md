# MapeadorDeRacks
Para mapear Racks De Redes - Switchs x Patch Panels

A Empresa precisa de um mapeador onde cada saída do Switch vai para uma entrada no Patch Panel
E cada Saída desse Switch está com uma VLAN Tagged ou Várias VLans General / Trunk
Para fins de mapeamento e não haver necessidade de abrir o Rack todo manualmente criei esse
sistema front-end simples mapeador de saídas para ajudar a empresa.

Para configuar as saídas do switch em suas devidas entradas no patch cord, você só precisa adicionar ou modificar
as já existentes pelos formulários e botões criados.

Exemplo: Após clicar sobre a saída 5 do Switch ela conecta o cabo de rede na entrada do patch panel 3 na entrada 37
e se voce clicar de novo em cima da saída 5 do Switch ela some para deixar a tela livre e limpa para você clicar em outra porta
